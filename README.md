

# 🩺 Curelio - Machine Learning Based Cashless Health Insurance Eligibility Prediction System

## 📌 Overview

Curelio is a Machine Learning based healthcare insurance prediction system developed as an internship project provided by DEVSPRINT.

The system is designed to predict cashless health insurance eligibility by analyzing healthcare and insurance-related information such as insurance provider, hospital details, policy type, and treatment details.

Curelio combines Machine Learning and Web Development technologies to provide a simple and efficient prediction platform through a Flask-based web application.

---

## 🚀 Features

- Predicts cashless health insurance eligibility
- Machine Learning based decision support
- User-friendly web interface
- Real-time prediction using trained ML model
- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- Multiple ML algorithm comparison
- Flask web application integration

## 🧠 Selected Model

After training and evaluating multiple machine learning algorithms, the best-performing model was selected for final implementation.

**Final Model: Decision Tree Classifier**

The trained model is stored using Joblib and integrated into the Flask application for real-time predictions.
